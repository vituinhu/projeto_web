Listas não ordenadas

A tag ul (unordered list - lista nao ordenada) representa uma lista de itens sem ordem rígida, por exemplo ingredientes em uma receita.

A tag li (list itens - itens da lista) representa um item que faz parte de uma lista.

Exemplo de lista

  • Primeiro item
  • Segundo item
  • Terceiro item
    • subitem

Listas ordenadas

A tag ol (ordered list - lista ordenada) representa uma lista de itens numerados sequencialmente.

Exemplo de lista ordenada

  1. primeiro item
  2. segundo item

Atributos

reversed (ordem reversa)
Exibe os itens da lista ordenada em ordem reversa

  1. item mais importante
  2. item menos importante

start (início)
específica o valor inicial para a numeração da lista

  1. Item 5
  2. Item 6

type (tipo)
indica o tipo de numeração: a - indica letras minúsculas A - indica letras maiúsculas i - indica algarismos romanos minúsculos I - indica algarismos romanos maiúsculos

  1. primeiro item
  2. segundo item
